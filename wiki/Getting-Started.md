# Getting Started

This guide will help you get started with Silence of the Lags speed test.

## Using the Web Version

### Step 1: Open the Application
Visit [https://anacondy.github.io/3-SOTL-Speed-test/](https://anacondy.github.io/3-SOTL-Speed-test/) in any modern web browser.

### Step 2: Check Connection Status
Look for the connection indicator at the bottom of the home screen. A green dot indicates you're connected to the internet.

### Step 3: Start the Test
Click the "INITIATE TEST" button to begin the speed test.

### Step 4: Wait for Results
The test will measure:
1. **Ping/Latency** (~2 seconds)
2. **Download Speed** (~8 seconds)
3. **Upload Speed** (~6 seconds)

### Step 5: View Results
Once complete, you'll see:
- Download speed (Mbps)
- Upload speed (Mbps)
- Ping (ms)
- Jitter (ms)
- Packet loss (%)
- Download time estimates

## Installing Desktop Applications

### Windows
1. Download the `.exe` file from [Releases](https://github.com/anacondy/3-SOTL-Speed-test/releases)
2. Run the installer
3. Follow the installation wizard
4. Launch from Start Menu or Desktop shortcut

### macOS
1. Download the `.dmg` file from [Releases](https://github.com/anacondy/3-SOTL-Speed-test/releases)
2. Open the DMG file
3. Drag the application to your Applications folder
4. Launch from Applications or Spotlight

### Linux
1. Download the `.AppImage` file from [Releases](https://github.com/anacondy/3-SOTL-Speed-test/releases)
2. Make it executable: `chmod +x SilenceOfTheLags.AppImage`
3. Run the AppImage: `./SilenceOfTheLags.AppImage`

## Installing Mobile Applications

### Android
1. Download the `.apk` file from [Releases](https://github.com/anacondy/3-SOTL-Speed-test/releases)
2. Enable "Install from Unknown Sources" in Settings
3. Open the APK file
4. Tap "Install"
5. Launch from app drawer

## Tips for Accurate Results

1. **Close other applications** using bandwidth
2. **Use a wired connection** if possible for desktop tests
3. **Disable VPN** for accurate speed measurements
4. **Test at different times** of day for comparison
5. **Run multiple tests** and compare results
