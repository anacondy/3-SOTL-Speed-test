# Supported Platforms

Silence of the Lags is designed to work across all major platforms and devices.

## Web Browser Support

### Fully Supported Browsers

| Browser | Minimum Version | Notes |
|---------|-----------------|-------|
| Google Chrome | 80+ | Best performance |
| Mozilla Firefox | 75+ | Excellent support |
| Apple Safari | 13+ | Full iOS/macOS support |
| Microsoft Edge | 80+ | Chromium-based |
| Opera | 67+ | Full feature support |
| Brave | 1.0+ | Privacy-focused |
| Vivaldi | 3.0+ | Power user features |

### Limited Support

| Browser | Notes |
|---------|-------|
| Internet Explorer 11 | Basic functionality only |
| Older Safari (< 13) | Some animations may not work |

## Accessibility Features

- **Keyboard Navigation**: Full keyboard support
- **Screen Readers**: ARIA labels and live regions
- **Reduced Motion**: Respects `prefers-reduced-motion`
- **High Contrast**: Enhanced borders and contrast
- **Large Text**: Scalable typography

## Network Requirements

- Active internet connection
- Minimum bandwidth: Any (the test measures your speed)
- Firewall: Allow HTTPS connections to CDN servers
- Proxy: May affect accuracy; direct connection recommended
